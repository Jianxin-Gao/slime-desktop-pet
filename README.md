# 👾 Slime Desktop Pet (史莱姆桌宠)

这是一个基于 Node.js + Electron 开发的桌面宠物应用。
它可以脱离浏览器，直接悬浮在你的电脑桌面上，支持拖拽移动、点击互动，陪你一起摸鱼！

## ✨ 核心功能
- **无边框透明窗口**：完美融入桌面背景。
- **互动气泡**：左键点击史莱姆，它会随机弹出一句话和你互动。
- **自由拖拽**：按住鼠标左键即可把它拖到屏幕的任意角落。
- **右键菜单**：支持右键菜单进行“喂食”、“摸摸”以及“退出桌宠”。

## 🚀 安装与运行指南

### 🍎 对于 macOS 用户

1. 前往本项目的[Releases 页面](这里替换成你的GitHub_Releases链接) 下载最新的 `.dmg` 安装包。
2. 双击打开 `.dmg` 文件，将 `SlimePet.app` 拖入你的 `Applications`（应用程序）文件夹。
3. **首次打开注意事项**：因为是个人开发者打包，Mac 可能会提示“无法验证开发者”。
   - **解决办法**：在 `应用程序` 文件夹中找到该 App，**按住键盘上的 `Control` 键不放，点击鼠标左键**，选择“打开”，在弹出的窗口中再次点击“打开”即可。

### 🪟 对于 Windows 用户

我们为 Windows 用户提供两种运行方式，任选其一即可：

#### 方法一：使用 `.exe` 直接安装（推荐）
1. 前往本项目的[Releases 页面](这里替换成你的GitHub_Releases链接) 下载最新的 `.exe` 安装包（或便携免安装版）。
2. 下载后直接双击运行即可。

#### 方法二：从源码本地运行（绿色极客版）
如果你不想安装，或者 `.exe` 文件被杀毒软件误报，你可以直接运行源码：
1. 下载并安装 [Node.js](https://nodejs.org/)（推荐下载 LTS 版本）。
2. 点击本仓库右上角的 `Code -> Download ZIP`，下载项目源码并解压。
3. 打开命令行终端（CMD 或 PowerShell），进入解压后的目录。
4. 依次运行以下命令：
   ```bash
   # 安装运行环境依赖
   npm install

   # 启动桌宠
   npm start
   ```

## 🛠 开发与打包说明

如果你想自己修改代码并打包，可以参考以下命令：

```bash
# 安装依赖
npm install

# 本地调试运行
npm start

# 打包 macOS 版本 (.app / .dmg)
npm run pack:mac

# 打包 Windows 版本 (.exe)
npm run pack:win
```
